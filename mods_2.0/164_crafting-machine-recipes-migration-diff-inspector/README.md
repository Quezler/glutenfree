I am currently working on integrating Space Exploration with Space Age.

Space Age adds a bunch of "-or-" recipe categories which pull recipes from categories that previously had them.

This inspector mod helps mod developers with finding out which recipes were added-to or removed-from a machine since the previous save.

Usage:
1) save a world with this mod installed
2) add/remove/edit a mod that moves recipes around
3) load that recently saved world again
4) /crafting-machine-recipes-migration-diff-inspector
